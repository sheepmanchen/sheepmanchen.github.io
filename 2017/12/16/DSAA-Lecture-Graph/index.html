Breadth First Search

  • SSSP(single source shortest path as an example): Shortest Path(unit weight: least edges), e.g., v1 to vk, length of path: k-1
  • BFS can solve this problem in O(V+E) time.

BFS Algorithm

Pseudocode 

· create a queue Q. Insert the source vertex s into Q. Mark s visited.  
· while Q is not empty  
    · De-queue v from Q (note: the first vertex)
    · For every out-neighbor u of v that visit[u]==false
        1. Enqueue u into Q, visit[u]=true;
        2. *//Make u a child of v in the BFS tree T.*  
· end
  • Note: When we finish this, there may still remain vertex that hadn’t been visited, which means that they cannot be reached from the source vertex.
  • Back to the SSSP problem: the shortest path from the source vertex a to any vertex x is exactly the path from a to node x in the BFS tree we construct.

Directed Cyclic Graph(DAG)

DAG: contains no cycles, e.g., spark, tensorflow

The Cycle Detection Problem

G=(V,E) be a directed graph, determin whether it is a DAG.
This problem can be solved in O(V+E) time by using DFS algorithm.

DFS algotithm

output a tree, DFS-tree
Pseudocode 

Create a Stack S. Pick an arbitrary vertex v. Push v into S. Mark v visited.
While (!S.empty())
    int v = S.front();//don't pop it!
    Q: V has an unvisited out-neighbor?
        1. Yes: let it be u
                Push u into S, visited[u]=true;
                Make u a child of v in the DFS-tree T;
        2. NO: pop v from S;
end while
If there still exist unvisited vertices, repeat the above by restarting fot an arbitrary unvisited vertex v'. Creating a new DFS tree rooted at v'.

Time complexity: O(V+E)
